Пусть $C$ — [циклический код](Циклический%20код.md) [длины](Длина%20кода.md) $p^m - 1$ над $F_{p^m}$ (или над $F_p$) с [порождающим многочленом](Порождающий%20многочлен%20циклического%20кода.md) $g(x)$ таким, что
$$\exists \ b \geqslant 0, \ \delta \geqslant 2 \ : \ g(\alpha^b) = g(\alpha^{b + 1}) = \dots = g(\alpha^{b + \delta - 2}) = 0$$
Тогда [минимальное расстояние](Кодовое%20расстояние.md) $d$ больше или равно $\delta$, т.е. $d \geqslant \delta$.

- $\alpha$ — [примитивный элемент](Примитивный%20элемент%20поля%20Галуа.md) из $GF(p^m)$
- $\delta$ — конструктивное расстояние кода $C$
