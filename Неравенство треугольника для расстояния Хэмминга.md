Для [расстояния Хэмминга](Расстояние%20Хэмминга.md) выполнено неравенство треугольника:
$$d(x, y) + d(y, z) \geqslant d(x, z)$$
для любых $x, y, z \in E_q^n$[^1]

[^1]: [Множество слов длины n над алфавитом из q элементов](Множество%20слов%20длины%20n%20над%20алфавитом%20из%20q%20элементов.md)
