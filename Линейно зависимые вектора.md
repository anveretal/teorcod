Вектора $\nu_1, \nu_2, \dots, \nu_n$ называются **линейно зависимыми**, если существует набор коэффициентов
$$( \alpha_1, \alpha_2, \dots, \alpha_n ) \neq ( 0, 0, \dots, 0 )$$
такой, что
$$\alpha_1 \nu_1 + \alpha_2 \nu_2 + \dots + \alpha_n \nu_n = 0$$

> [!NOTE] Другими словами
> Вектора называются **линейно зависимыми**, если есть нетривиальная линейная комбинация, равная нулю.
