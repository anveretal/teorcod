Это две ключевые функции для декодирования.

Для вещественного вектора $s = (s_0, \ldots, s_{N-1})$:

$$f(s) := \bigl(f(s_0, s_{N/2}),\ \ldots,\ f(s_{N/2-1}, s_{N-1})\bigr)$$

где поэлементно: $$f(a, b) = \text{sign}(ab)\cdot\min(|a|, |b|)$$

Для вектора $s$ длины $N$ и двоичного вектора $x$ длины $N/2$:

$$g(s, x) := \bigl(g(s_0, s_{N/2}, x_0),\ \ldots,\ g(s_{N/2-1}, s_{N-1}, x_{N/2-1})\bigr)$$

где поэлементно: $$g(a, b, x) = \begin{cases} b + a, & x = 0 \\ b - a, & x = 1 \end{cases}$$
